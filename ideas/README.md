# SFBI — Science Fiction Book Ideas

A two-volume concept dossier: 100 original novel premises, each with a one-paragraph synopsis and a three-act outline. Every entry is a seed, not a contract — any of them can be expanded into a full treatment (chapter map, character sheets, science memo).

## What's in each volume

**Volume I — Fifty Signals** (far-future, cosmic scale): hard-science survival, cosmic horror, apocalyptic and post-apocalyptic worlds, first contact & vast objects, wild cards. Filterable by subgenre.

**Volume II — Fifty Mirrors** (near-present, 2026–2045): our world plus one disciplined speculative element per premise, held to a literary standard. Six drawers — the mind observed; kin; the near sky; slow collapse; the machine question; last things. Stakes deliberately vary from a single marriage to the whole sky; each entry names, in one italic line, the existential question the book is secretly about. Filterable by theme and by scale (intimate / community / global).

The HTML files are self-contained (no external dependencies) — open them directly in a browser, light and dark theme both supported.
